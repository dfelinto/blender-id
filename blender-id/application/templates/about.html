What is the Blender ID?

The Blender ID is a unified login system that will give you access to Blender Institute and Blender Foundation web platforms.

With just one account you can set up a Blender Cloud subscription, manage your BFCT application, your Blender Conference talks, and much more in the future.

What is/will be using the Blender ID:

  • Blender Cloud
  • Blender Store
  • Blender Network (soon)
  • Blender Conference (soon)
  • Blender Development Fund (soon)
  • Blender Foundation Certified Trainer Program
